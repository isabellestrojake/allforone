
# All for One

Comandos básicos muito utilizados no SQL.


## Funcionalidades

- SQL
- NorthWind
- Docker


## Desenvolvimento

Foi programado através de `Docker`

## Instalação

Faça o `git clone`

```bash
  git clone git@github.com:isabellestrojake/allforone.git
  cd allforone
```

Instale as dependências com npm

```bash
  npm install
```
